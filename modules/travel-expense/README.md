# 差旅费报销工具

## 工具说明

本工具用于自动处理差旅费报销相关文件，通过OCR识别技术提取关键信息，自动生成报销所需的全部文档。

## 版本说明

### 单接口版本 (travel-expense-single.js)
- **特点**: 一次性上传所有文件，系统自动分类识别
- **适用场景**: 文件数量较少，希望快速处理
- **上传方式**: 拖拽或点击上传，支持批量选择

### 多接口版本 (travel-expense-multi.js)
- **特点**: 按类别分别上传文件，分类管理
- **适用场景**: 文件数量较多，需要精确控制
- **上传方式**: 8个分类接口，分别上传对应类型文件

## 文件分类

| 分类 | 说明 | 支持格式 |
|------|------|----------|
| 🏨 住宿费 | 酒店发票、住宿支付截图 | PDF, XML, JPG, PNG |
| ⛽ 燃油费 | 加油发票、燃油支付截图 | PDF, XML, JPG, PNG |
| 🚗 租车费 | 租车发票、结算单、支付截图 | PDF, XML, JPG, PNG |
| 🛣️ 过路费 | 过路费发票、通行支付截图 | PDF, XML, JPG, PNG |
| 🚄 高铁/火车票 | 高铁票、火车票 | PDF, XML, JPG, PNG |
| 📋 审批表 | 外出办案审批表 | PDF, JPG, PNG |
| 📄 立案报告 | 案件立案报告 | PDF |
| 📊 结算单 | 租车费用结算单 | PDF, JPG, PNG |

## 生成文档

工具会自动生成以下6个文档：

1. **业务事项审批单** (Excel) - 包含事项名称、金额、说明等
2. **伙食补助费领取表** (Excel) - 参与人员、天数、补助金额
3. **租车费用审批明细表** (Excel) - 按车牌号分别生成
4. **梳理表** (Excel) - 各类费用按日期统计
5. **出差费用说明** (Word) - 案件情况、费用明细
6. **配合调查案件说明** (Word) - 配合调查情况说明

## 安装部署

### 方法一：管理员上传（推荐）

1. 登录工具平台管理员界面
2. 进入"工具管理" → "上传新工具"
3. 选择对应版本的JS文件上传：
   - 单接口版本：`单接口版本/travel-expense-single.js`
   - 多接口版本：`多接口版本/travel-expense-multi.js`
4. 填写工具信息：
   - 工具ID：`travel-expense-single` 或 `travel-expense-multi`
   - 工具名称：差旅费报销工具（单接口版/多接口版）
   - 分类：财务工具
5. 保存并启用工具

### 方法二：GitHub仓库部署

1. 将文件复制到GitHub框架的modules目录：
   ```
   modules/
   └── travel-expense/
       ├── 单接口版本/
       │   └── travel-expense-single.js
       ├── 多接口版本/
       │   └── travel-expense-multi.js
       └── modules.json
   ```

2. 更新主配置文件 `config/modules.json`，添加工具配置：
   ```json
   {
     "modules": [
       {
         "id": "travel-expense-single",
         "name": "差旅费报销工具（单接口版）",
         "entry": "travel-expense/单接口版本/travel-expense-single.js"
       },
       {
         "id": "travel-expense-multi",
         "name": "差旅费报销工具（多接口版）",
         "entry": "travel-expense/多接口版本/travel-expense-multi.js"
       }
     ]
   }
   ```

3. 提交并推送到GitHub仓库
4. 等待GitHub Pages自动部署

## 使用步骤

### 单接口版本

1. 打开工具，进入上传界面
2. 拖拽或点击上传所有文件
3. 系统自动分类并显示文件列表
4. 点击"开始处理"按钮
5. 等待OCR识别和数据处理
6. 查看识别结果
7. 下载生成的文档

### 多接口版本

1. 打开工具，查看8个分类卡片
2. 逐个分类上传对应文件：
   - 点击分类卡片的"上传文件"按钮
   - 或拖拽文件到对应区域
3. 上传完成后查看汇总信息
4. 点击"开始处理"按钮
5. 等待分类识别和数据处理
6. 查看识别结果
7. 下载生成的文档

## 数据提取规则

### 业务事项审批单
- **事项名称**: 赴"到达地"调查案件差旅费
- **部门名称**: 专卖科
- **金额**: 全部发票金额 + 伙食补助费
- **事项说明**: 案件情况 + 期间产生的各项费用

### 伙食补助费领取表
- **参与人员**: 出差审批单中的出差人（多选）和行业外人员
- **所属单位**: 行业内人员填"睢阳区烟草局"，行业外人员填实际单位
- **起止日期**: 
  - 有高铁票：按高铁票最早/最晚时间
  - 无高铁票：按所有发票开票日期和支付记录日期的最早/最晚
- **天数**: 开始当天 + 结束当天 + 中间天数
- **补助标准**: 100元/天
- **补助金额**: 天数 × 100 × 人数

### 租车费用审批明细表
- **单位**: 睢阳区局
- **租车起止日期**: 结算单中的实际取车/还车时间
- **租车原因**: 赴"办案地点"调查"案件名称"
- **车辆信息**: 从结算单提取（车辆名称、车牌号、排气量）
- **费用明细**: 按日期显示加油数量、燃油费、过路费
- **表格命名**: 按车牌号码命名

### 梳理表
- **高铁费**: 按日期分别显示票价、人数、合计
- **住宿费**: 按日期显示发票金额
- **燃油费**: 按日期显示金额
- **租车费**: 显示租车费用合计
- **过路费**: 按日期显示金额
- **补助费**: 显示天数、人数、标准、合计
- **汇总**: 各项费用总计

### 出差费用说明（Word）
- **标题**: 赴"到达地"出差费用的说明
- **第一段**: 案件情况（从立案报告提取）
- **第二段**: 租车情况（如有租车）
- **第三段**: 各类费用合计
- **落款**: 专卖科 + 日期（出差最后一天的后一天）

### 配合调查案件说明（Word）
- **标题**: 关于配合烟草部门调查案件的说明
- **内容**: 案件编号、案件情况、配合调查人员、调查时间地点
- **落款**: 联合办案单位（行业外）+ 日期

## 技术说明

### OCR识别
- 使用 Tesseract.js 进行离线OCR识别
- 支持中文识别
- 识别准确率约85-95%

### 文件处理
- PDF解析：使用pdfjs-dist
- Excel生成：使用ExcelJS
- Word生成：使用docx.js

### 数据匹配
- 发票与支付记录通过金额匹配
- 过路费通过车牌号码匹配
- 日期范围自动计算

## 注意事项

1. **文件质量**: 请确保上传的文件清晰可读，模糊或倾斜的图片可能影响识别准确率
2. **文件命名**: 建议按类型命名文件，有助于提高自动分类准确率
3. **数据核对**: 生成的文档请务必核对关键信息（金额、日期、人员等）
4. **浏览器支持**: 建议使用Chrome、Edge等现代浏览器
5. **文件大小**: 单个文件不超过10MB

## 故障排除

### 文件上传失败
- 检查文件格式是否支持
- 检查文件大小是否超过限制
- 尝试刷新页面重新上传

### OCR识别失败
- 确保图片清晰，文字可读
- 避免图片过度倾斜或反光
- 尝试重新上传更清晰的图片

### 数据提取错误
- 手动核对提取的信息
- 在生成文档前修改错误数据
- 联系管理员更新识别规则
